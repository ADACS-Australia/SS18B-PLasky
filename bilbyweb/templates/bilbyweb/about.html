Welcome to Bilby: The user-friendly Bayesian inference library for gravitational-wave astronomy.
Bayesian parameter estimation is fast becoming the language of gravitational-wave astronomy. It is the method by which gravitational-wave data is used to infer the sources astrophysical properties. Bilby is a Bayesian inference code designed to do parameter estimation for gravitational-wave astronomy. This front end combines expert-level parameter-estimation infrastructure with a simple-to-use interface that facilitates use by beginners and experts alike.
